Yokiy 运维自动化平台项目

项目地址：https://gitee.com/jdytrip/jdy_devops.git

项目简介

  • 这是一个web自动化平台，为企业实现一键部署。
  • 基于Python 3 和 Django2.0
  • Bootstrap ampleadmin 模板
  • Docker kubernetes
  • Mysql 5.6 & Redis 4.0

主要功能

  1. 用户权限管理
  2. CI/CD 持续部署，持续构建
  3. MySql执行管理
  4. 应用日志管理
  5. 应用监控
  6. API监控
